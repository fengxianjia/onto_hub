# Changelog

所有对 **OntoHub** 项目的显著更改都将记录在本文件中。

## [1.2.0] - 2026-02-09

### Added
- **本体管理面板**：在本体详情对话框中新增“管理设置”页签。
- **元数据编辑**：支持动态修改本体系列的显示名称和默认解析模板。
- **数据重析功能**：支持针对特定版本触发手动重新解析任务，允许在重析时指定解析模板。
- **危险操作二次确认**：为数据重析增加了带有详细版本信息的二次确认提示。

### Changed
- **对话框 UI 优化**：将所有弹窗布局调整为响应式 `vw` 单位，提升在大屏幕上的视觉体验。
- **页签导航增强**：优化了详情页的导航逻辑，重析完成后会自动引导至图谱或列表视图。
- **组件命名规范**：将前端 `Input` 和 `Select` 组件重命名为 `UIInput` 和 `UISelect` 以避免与原生 HTML 标签冲突。

### Fixed
- **后端进程卡死修复**：解决了多个僵尸 Python 进程导致的接口挂起（Pending）且无日志的问题。
- **Select 交互失效**：修复了因组件命名冲突导致的下拉框无法点击的问题。
- **UI 错位**：修复了管理设置中输入框标签丢失及排版不齐的问题。

## [1.1.0] - 2026-02-07

### Added
- **Webhook 推送详情**：新增推送状态展示对话框。
- **原子化删除逻辑**：实现了防误删保护机制（Active 版本及正在使用的版本不可删除）。

### Changed
- **设计系统集成**：全量接入了极简现代主义设计系统，包括品牌渐变色（Antigravity Blue）和新的排版规范。
